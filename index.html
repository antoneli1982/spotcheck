Controle de Spot Check

Ferramenta em HTML com importação via Excel para leitura, tratamento e monitoramento de Spot Check. O painel cruza o código do fornecedor com a base de cadastro, mostra quantidade de checks por local, quantidade de divergências e o ranking por fornecedor em volume absoluto e relativo.

Excel como entrada Layout no design system futurista/logístico Fornecedor pelo código Dash para operação e gestão
Aceita .xlsx, .xls ou .csv. Para XLSX, a biblioteca SheetJS é carregada via CDN. Arraste a planilha para esta área ou clique para selecionar.
